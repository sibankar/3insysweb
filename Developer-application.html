About our Senior .Net Full Stack Developer

We are currently seeking a Senior .Net Full Stack Developer to join our team in downtown Los Angeles in a beautiful building and convenient location!

As Senior .Net Full Stack Developer will be responsible for, but not limited to, the following: reviewing architectural and technical requirements, designing and developing enterprise solutions, focusing on solution development throughout the project life-cycle, supporting development and testing activities, supporting best practices and proven patterns and executing proofs of concept on various technologies.

Required Qualifications:
  • • 5+ years of hands-on experience in designing and developing SaaS
  • • 2+ years experience with MVC and MVVM architecture design
  • • Experience in creating and using scalable and distributed RESTful Web API
  • • Experience with various SaaS architecture designs - Single DB, Isolated DB, Hybrid DB / Single and Dynamic Themes
  • • Experience in developing modern web sites, dashboards and visualizations
  • • Working knowledge with test-driven development
  • • Experience with Asynchronous, Parallel, Reactive programming paradigms
  • • Strong product and project management skills
  • • Participate in product development process of SaaS architecture and applications
  • • Commitment to design scalable, secure and reusable software
  • • Strictly employ Development/Test/Acceptation/Production/Backup (DTAPB) environments in the software development cycle
  • • Lead, collaborate, mentor and manage small team of developers (onshore/offshore)
  • • Work independently, to meet deadlines, adapt to shifting priorities, and manage multiple projects at a time
  • • Review code and promote/enforce clean coding principles and design patterns
  • • Excellent presentation, verbal and written communication skills
  • • Ability to communicate both technical and non-technical aspects in clear and effective mannerS or higher in Computer Science or related technical discipline
Required Skills:
  • • Web Application Framework: ASP.NET Core, ASP.NET Core MVC
  • • Programming Languages: C#, SQL
  • • JavaScript Libraries / Frameworks: JQuery, AngularJS, ReactJS
  • • CSS Frameworks: Bootstrap 4
  • • Database Technologies: MySQL 5.7 / 8, MS SQL Server 2017 / 2019
  • • Object-Relational Mapper (ORM): EntityFramework Core
  • • Collaborative Code Repositories: Git, GitHub, Team Foundation Server
  • • Virtual Machine / Containers: Hyper-V, Docker, Kubernetes
  • • Content Management System (CMS): Any on-site or SaaS
Preferred Skills:
  • • Integrate 3rd party API : Slack, MailChimp, Twilio, Paypal, Stripe, Google, DropBox, Facebook, Twitter, Instagram, Pinterest and others
  • • Cloud services - Design, Development, Deployment and Maintain
  • • Server-side technologies - Nginx, Redis, Apache, ElasticSearch, etc
  • • Security technologies - SSO, IDM, SAML, OAuth, JWT
  • • Experience of deploying and managing infrastructure in a data center
